# software_engineering

> **big_work**

## 简介

本项目是一个融合多种前端和后端技术的综合性工程实践，主要用于软件工程课程实践。项目采用了主流的技术栈，旨在提升团队成员的软件开发能力和项目协作经验。

## 技术栈

- **前端：**
  - JavaScript
  - Vue.js
  - CSS
  - HTML
- **后端：**
  - Java

## 目录结构（示例）

```
.
├── frontend/             # 前端代码（JavaScript、Vue、CSS、HTML）
├── backend/              # 后端代码（Java、配置等）
├── docs/                 # 文档资料
├── README.md             # 项目说明文档
└── ...
```

## 快速开始

1. **克隆项目**

   ```bash
   git clone https://github.com/yzz23270140/software_engineering.git
   cd software_engineering
   ```

2. **前端运行**

   - 进入 `frontend` 目录后，安装依赖并启动项目

   ```bash
   cd frontend
   npm install
   npm run serve
   ```

3. **后端运行**

   - 进入 `backend` 目录后，根据实际情况使用 Java 环境进行编译和运行

   ```bash
   cd backend
   # 比如使用 Maven 或者 Gradle 进行编译启动
   ```
