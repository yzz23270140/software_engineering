This project is a comprehensive engineering practice integrating various front-end and back-end technologies, mainly used for software engineering course projects.   - Vue.js
  - CSS
  - HTML
- **Backend:**

## Quick Start

1. **Clone the Project**

   ```bash
   git clone https://github.com/yzz23270140/software_engineering.git
   cd software_engineering
   ```

2. **Run Frontend**

   - Go into the `frontend` directory, install dependencies, and start the project

   ```bash
   cd frontend
   npm install
   npm run serve
   ```

3. **Run Backend**

   - Go into the `backend` directory and use a Java environment for compilation and startup as needed

   ```bash
   cd backend
   # For example, use Maven or Gradle to build and run
   ```
